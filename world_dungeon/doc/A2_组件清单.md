## 附录B：组件完整清单

所有组件的名称和用途，方便实现时对照：

```
空间类：
  position          → 实体当前所在room_id
  room_info         → 房间类型/楼层/模板（Room专用）
  connections       → 房间连接表（Room专用）
  room_state        → 房间光照/标签/探索状态（Room专用）

行动类：
  action_timer      → cooldown_cur / cooldown_max / ready
  ai_behavior       → archetype名称 + 参数
  ai_intent         → 当前tick的行动意图（临时，执行后清除）
  adventurer_goal   → 冒险者目标状态机（冒险者专用）

属性类：
  stats             → hp/hp_max/attack/defense/speed/level
  actor             → move_cooldown / attack_cooldown
  active_modifiers  → 所有活跃的modifier
  faction           → 阵营id + 对其他阵营的态度

战斗类：
  in_combat         → 当前交战目标集合 + 进入战斗tick

RPG类：
  level             → current/xp/xp_next
  skills            → { skill_id → { level, cooldown_cur } }
  equipment_slots   → 各装备槽的item_eid
  inventory         → 背包容量/当前重量/物品集合
  loot_table        → 死亡后掉落表

物品类：
  item_info         → 名称/标签/重量/价值/堆叠信息
  location          → 物品当前位置（地面/背包/装备）
  equippable        → 装备槽位 + 提供的modifiers
  weapon_data       → 伤害类型/伤害骰/命中特效
  usable            → 使用效果/目标类型/是否消耗

标记类：
  definition        → 来源def_id（调试用）
  unique            → 标记该实体是全局唯一的
  is_room           → 标记该实体是房间（方便query过滤）
  is_actor          → 标记该实体是可行动的生物
  is_item           → 标记该实体是物品
```
